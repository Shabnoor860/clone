# Vistagram

Vistagram is a full-stack social media web application inspired by modern platforms.

## 🚀 Live Demo
https://vistagram-grlu.onrender.com

## 🛠 Tech Stack
- Python (Flask)
- SQLAlchemy
- Flask-Login
- Gunicorn
- SQLite
- HTML
- CSS
- JavaScript

## ✨ Features
- User Authentication
- Create Posts
- Like & Comment System
- Stories (24h expiry)
- Follow / Unfollow
- Direct Messaging
- Notifications
- Profile Editing
- Image Uploads

## ⚙ Deployment
Deployed on Render using Gunicorn production server.
